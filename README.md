# 🚕 Taxi Mate (Taxi-Sharing Matching Service)
> **"함께 타고, 함께 아끼세요"** > 대학생들의 등하교 교통비 부담을 줄이고 편리한 이동을 돕는 택시 동승자 매칭 서비스

---

<br>

## 프로젝트 소개
- **서비스명:** Taxi Mate
- **핵심 가치:** 비용 절감, 이동 편의성, 신뢰 기반 동승
- **주요 기능:**
  - **실시간 핀 생성 및 검색:** 원하는 시간과 장소에 동승자 모집 핀 생성
  - **실시간 채팅:** 매칭된 인원 간의 원활한 소통 (Django Channels 활용)
  - **OCR 정산 시스템:** 영수증 인식을 통한 투명한 1/N 정산 및 카카오페이 송금 연동
  - **사용자 신뢰도:** 매너 온도 및 태그 시스템으로 안전한 이용 보장
